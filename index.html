    mica-mqtt

    基于 java aio 实现，开源、简单、易用、低延迟、高性能百万级 java mqtt client 组件和 java mqtt broker 服务。

    mqtt入门快速开始最佳实践

    简单、轻量、易集成、持续更新！

    • 支持 MQTT v3.1、v3.1.1 以及 v5.0 协议。

    • 支持 websocket mqtt 子协议（支持 mqtt.js）。

    • 支持 http rest api。

    • 支持 MQTT client、server 共享订阅。

    • 支持 MQTT 遗嘱消息。

    • 支持 MQTT 保留消息。

    • 支持 MQTT client、server 支持原生 Android。

    • 支持 Spring boot、Solon 和 JFinal 项目快速接入。

    • 支持 GraalVM 编译成本机可执行程序。
